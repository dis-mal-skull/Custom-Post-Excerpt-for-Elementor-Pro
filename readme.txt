To use this code, simply save it to a file called post-excerpt.php and place it in the wp-content/plugins/ directory of your WordPress site. Then, activate the plugin in the plugins section of your website and you should be able to use the shortcode [p-ex] on any post or page to display an excerpt of the post with the number of words you specify in the shortcode.


Shortcode:

[p-ex] = default 50 words
[p-ex words="20"] = custom 20 words

How to use:

Add a Text Editor Item to your Canva and switch the text editor to code and paste the shortcode, also you can edit the font size, color etc, in style tab.
